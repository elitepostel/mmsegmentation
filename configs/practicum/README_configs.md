# Configs for practicum experiments

Здесь храните конфиги всех экспериментов.

Рекомендуемый принцип:
- один файл конфига = один эксперимент;
- имя файла отражает суть изменения.

Примеры:
- `baseline_segformer_b0.py`
- `baseline_deeplabv3_r50.py`
- `exp01_stronger_aug.py`
- `exp02_ce_dice_loss.py`
- `exp03_larger_backbone.py`
